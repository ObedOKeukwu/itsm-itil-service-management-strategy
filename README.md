# ITSM Deployment Strategy Using ITIL

## Overview

This project presents an IT Service Management (ITSM) deployment strategy based on the Information Technology Infrastructure Library (ITIL) framework.

The objective was to design a practical ITSM implementation plan for a growing organization, focusing on service delivery, incident management, change management, user adoption, and continuous improvement.

## Key Topics

- ITIL Framework
- Incident Management
- Change Management
- Service Request Management
- Problem Management
- Service Delivery and Support
- Service Improvement
- Employee Training and Adoption
- IT Performance Metrics (KPIs)
